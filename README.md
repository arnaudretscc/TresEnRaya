# TresEnRaya
Created with CodeSandbox

## Primer proyecto creado con Capitole Consulting - Dia: 3/11/2022
Este proyecto cuenta de un modelo de tres en raya, solamente html y css. 
 ### Funcionalidad
  - Le das a jugar
  - Entonces te aparece para elegir jugador (X o O)
  - Seguidamante se elige al azar un jugador para empezar
  - Empieza el juego
  - Cuando el juego termina te puede aparecer (O Wins, X Wins, Draw)
